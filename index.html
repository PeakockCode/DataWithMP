Top Czech YouTubers
2024

In this project, we extracted, cleaned, prepared, and analyzed data from YouTube channels using Python (Jupyter Lab) and SQL (Microsoft SQL Server). Finally, we prepared documentation to summarize all the insights from our analysis and provided recommendations for the marketing department.
